## How to run

### Get access to some ROMs

There are public domain ROMs [here](https://www.zophar.net/pdroms/chip8.html).

### Run program
```shell
python pychip8/Emulator "ROM file"
```

## How to test

### Unit Tests
```shell
python -m unittest discover
```
